# Kicad-9.0-Plugin-K.I.R-Kicad.Ibom.Reorder
This plugin is a complementary plugin to the Interactive HTML BOM plugin. It is used to reorganize the file provided by the Interactive HTML BOM plugin.
## To install this plugin, you must:
  Open KiCad 
  Go to Plugins and Content Manager 
  Click on Install from file 
  And insert the .zip file named plugin_kir_v1.zip, which is located in plugin_kicad_KIR
