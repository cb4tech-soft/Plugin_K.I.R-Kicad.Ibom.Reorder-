# K.I.R-Kicad.Ibom.Reorder
This plugin is a complementary plugin to the Interactive HTML BOM plugin. It is used to reorganize the file provided by the Interactive HTML BOM plugin. This plugin is compatible with KiCad 9.0.
## Installation
  Open KiCad. Go to Plugins and Content Manager. Click on Install from file. And insert the .zip file named plugin_kir_v2.zip, which is located in  the release [plugin_kicad_KIR](https://github.com/cb4tech-soft/Plugin_K.I.R-Kicad.Ibom.Reorder-/releases/tag/release)
